# 🛒 Mercado App

Um aplicativo web completo para gerenciamento de compras de supermercado com carrinho, lista de planejamento e gestão de categorias e produtos.

## ✨ Funcionalidades

✅ **Tela Home** - Visualize categorias e produtos populares  
✅ **Lista de Planejamento** - Anote o que precisa comprar antes de ir ao supermercado  
✅ **Carrinho de Compras** - Adicione e remova produtos  
✅ **Gerenciamento Completo** - Crie, edite e exclua categorias e produtos  
✅ **Backup de Dados** - Exporte e importe seus dados em JSON  
✅ **Armazenamento Local** - Todos os dados são salvos no LocalStorage do navegador  
✅ **Design Responsivo** - Interface mobile-first simulando um smartphone  

## 🚀 Como Usar

1. Acesse o aplicativo
2. Navegue pelas 4 abas:
   - **Início**: Veja categorias e produtos
   - **Lista**: Planeje suas compras
   - **Carrinho**: Veja os itens selecionados
   - **Config**: Gerencie categorias, produtos e faça backup

## 🎨 Personalização

No menu **Config**, você pode:
- Alterar o nome da saudação
- Alterar o subtítulo
- Criar novas categorias
- Adicionar/editar/excluir produtos
- Fazer backup dos dados

## 💾 Backup e Restauração

- **Exportar**: Clique em "Exportar Dados" para baixar um arquivo JSON
- **Importar**: Clique em "Importar Dados" e selecione um arquivo JSON válido
